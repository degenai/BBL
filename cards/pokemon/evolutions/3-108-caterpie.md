## Trivia

- **Set context** — The XY Evolutions set (XY12) was released November 2, 2016, as a deliberate tribute to the original Pokemon Base Set (1998), timed to the franchise's 20th anniversary. The majority of its cards reproduce Base Set artwork using the original Ken Sugimori flat illustration style — including this Caterpie, which reproduces the original Base Set card. The set contained 108 cards plus secret rares, mixing straight reprints with new Mega Evolution and BREAK cards. `[PokemonTCG.io API: xy12-3; WebSearch: PokeBeach/Evolutions announcement]`
- **Artist context** — Ken Sugimori is the art director and lead character designer at Game Freak who designed the original 151 Pokemon. His flat, clean illustration style defined the visual aesthetic of the original TCG. The Evolutions set deliberately preserved his pre-digital rendering rather than updating to modern painted techniques. `[WebSearch: PokeBeach/Evolutions; PokemonTCG.io API]`
- **Flavor text** — "It is covered with a green skin. When it grows, it sheds the skin, covers itself with silk, and becomes a cocoon." This text matches the original Base Set flavor text. Caterpie is National Pokedex no. 10 and evolves into Metapod (no. 11), then Butterfree (no. 12). `[PokemonTCG.io API: xy12-3]`
