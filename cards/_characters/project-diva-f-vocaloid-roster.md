# Project DIVA F Vocaloid roster (Crypton + guest voicebanks)

The licensed-character cohort of named Vocaloid mascots — and voicebank-adjacent guest characters — depicted across SEGA's Project DIVA F rhythm game (2012 PS3 / 2013 PS Vita) and re-licensed into the Weiss Schwarz PD/S22 trading-card set (2014). The core roster is Crypton Future Media's Character Vocal Series — Hatsune Miku (CV01), Kagamine Rin and Len (CV02), Megurine Luka (CV03) — plus Crypton's earlier-era Yamaha-Vocaloid pair MEIKO and KAITO. The PD/S22 set additionally licenses guest voicebanks Kasane Teto (UTAU, not a Vocaloid proper) and Yowane Haku (Crypton-derivative fan-character formalized as a guest module). Every depicted figure is a designer-named virtual performer wearing a designer-named "module" (costume variant) from the Project DIVA F module catalog; the BBL graph can edge those module-printings together through this cohort node rather than card-to-card.

## What this node anchors

Card art and rules-text that depicts one of the Project DIVA F roster characters in one of the game's licensed modules (costume variants). Membership predicates on the figure being a Crypton-or-licensed-guest voicebank with a Vocaloid Wiki character page AND the depicted costume being identifiable as a Project DIVA F module (or a directly Crypton-licensed module variant) by name match or visual match. Generic Miku-vibe fan art is not sufficient; the figure must be a named voicebank in a SEGA-or-Crypton-licensed appearance.

**Cards in our corpus pointing here (current, enriched):**

- **Hatsune Miku"Factory Tyrant"** (PD/S22-E027 RR, Double Rare) — Miku in the *Factory Tyrant* module: dark structured punk-industrial jacket with choker-style collar, vivid red eyes (a hallmark identifier distinguishing this module from default-eye Miku printings), against a dark industrial-stage backdrop of angular geometric set pieces. The defiant register reads as the dark-punk counterpart to the Divine Goddess / Dark Angel ceremonial inversions.
- **"Together with You"Hatsune Miku** (PD/S22-E031 R, Rare) — Miku in the *Together with You* module: flowing pale-teal-and-white costume, open-palm forward reach toward the viewer, soft pink-and-teal sparkle background. The warm intimate-concert register contrasts directly with Factory Tyrant's defiance and is one of the connective / audience-facing modules in the Project DIVA F catalog.
- **Hatsune Miku"Divine Goddess"** (PD/S22-E033 U, Uncommon) — Miku in a celestial-goddess module: white-and-teal robe with white fur trim, large gold halo behind her, deep starfield background. The "Divine" / sublime register reads as one of the higher-rarity ceremonial modules in the Project DIVA F catalog.
- **Hatsune Miku"Deep Sky"** (PD/S22-E041 C, Common) — Miku in an azure-and-navy stage module with celestial accents, deep-sky palette.
- **Weekender Girl** (PD/S22-E046 C, Common) — Miku appearance tied to the *Weekender Girl* song module by kz (livetune); the song is a canonical Project DIVA F track and the depicted character is Miku.
- **Hatsune Miku"Dark Angel"** (PD/S22-E067 C, Common) — Miku in a black-feathered fallen-angel module, the dark inversion of the Divine Goddess register.
- **Melancholic** (PD/S22-E021 C, Common) — depicts Kagamine Rin in the *Melancholic* song module (Junky's *Melancholic* is a canonical Vocaloid track originally sung by Kagamine Rin); short blond hair, green-and-yellow costume, headphones. The card title is the song name; the depicted figure is Rin.
- **Kagamine Rin"Append"** (PD/S22-E014 U, Uncommon) — Rin in the *Append* module: white-and-black mechanical-transparent aesthetic (replacing the original yellow-and-black) corresponding to the Append voicebank release (Crypton, December 27, 2010) which added Power, Warm, and Sweet vocalization styles to the original Rin voicebank. The visual aesthetic was designed by Wataru Sasaki to convey "a sense of transparency in the heart." The card's Level-2 Change effect explicitly retrieves Kagamine Rin "Original" from the waiting room, modeling the voicebank-extension relationship at the rules layer.
- **Kagamine Rin"School Swimwear"** (PD/S22-E087 U, Uncommon) — Rin in a Project DIVA F school-swimwear module.
- **Kagamine Len"Original"** (PD/S22-E002 RR, Double Rare) — Len in his *Original* / default module: white-and-black sailor-collar uniform with yellow scarf accent, holding a flat device, against a warm amber-orange radiating-burst backdrop. The Original module is the canonical baseline appearance for Len's voicebank-character identity and is the reference design from which subsequent module variants derive.
- **Kagamine Len"Trickster"** (PD/S22-E017 C, Common) — Len in the *Trickster* module variant.
- **Kagamine Len"Bad Boy" (A)** (PD/S22-E084a U, Uncommon) — Len in a red-and-black geometric-jacket module with headphones; the (A) suffix indicates a Weiss Schwarz dual-art printing.
- **MEIKO"Blue Crystal"** (PD/S22-E056 R, Rare) — MEIKO in the *Blue Crystal* module; MEIKO is Crypton's original-era female voicebank (pre-CV-series, original Yamaha Vocaloid engine), distinct from the CV-series Miku/Rin/Len/Luka.

Thirteen enriched corpus members across five named voicebanks (Hatsune Miku x6, Kagamine Rin x3 including Melancholic and the Append voicebank-variant, Kagamine Len x3 including the Original baseline, MEIKO x1).

## The design thesis

The Project DIVA F roster's most BBL-relevant property is that every depicted figure is a *named virtual performer wearing a named costume variant* — Vocaloid characters are software products with designer-published canonical appearances, and Project DIVA F's "module" system formalizes the costume variants as a discrete catalog where each module has a name, an unlock condition, and a per-character compatibility list. The Weiss Schwarz PD/S22 set licenses both the characters AND the specific module variants, which means every card in the cohort is dual-anchored: to the voicebank-character (the IP layer) AND to the module (the costume-variant layer). This is a cleaner, more designer-formalized version of the multi-aspect character pattern that the existing `[[nicol-bolas]]` node uses to bridge War of the Spark printings — but where Bolas's "aspects" are story-arc beats, the Vocaloid roster's aspects are catalogued module variants from a published game's costume system.

The cohort is also a *cross-platform IP property* — the characters originate as Crypton-published singing-synthesizer voicebanks (software products with character mascots), get formalized as performance avatars across SEGA's Project DIVA rhythm-game series, and license through to Weiss Schwarz as collectible printings. The card art is SEGA-or-Crypton-supplied per the licensing chain, not new commissions. For BBL the cohort anchors a `[[labor]]` adjacency through the *idol-as-software-product* register (the performer is literally a synthesizer voicebank rented to producer-musicians; the music is collaborative-labor between Crypton, the voicebank user, and the audience that adopts the song) and a future Discrete Lair candidate built around the *virtual-idol-as-cooperative-labor-product* thesis: *the voice is software, the performer is licensed, the music is collaborative — and yet the audience treats the character as a person.*

## Caveats

- **Scope is bounded to the PD/S22 set, not the entire Vocaloid universe.** The Vocaloid software family includes dozens of named voicebanks across multiple publishers (Crypton, Internet Co., AH-Software, 1st Place, etc.) and hundreds of fan characters and derivative voicebanks. This node is deliberately scoped to *the roster licensed into SEGA's Project DIVA F and re-licensed into Weiss Schwarz PD/S22* — the same scoping principle as `[[eld-cinderella-cycle]]` (Throne of Eldraine set scope, not all Cinderella adaptations in MTG). If a future BBL acquisition brings non-PD/S22 Vocaloid cards into corpus (e.g. Project Sekai cards, a Hatsune Miku Bushiroad set, an IA / Vocaloid-3 set), a parent `crypton-vocaloid` or `vocaloid-roster` node should be authored above this one and this PD/S22 cohort re-parented under it.
- **Modeled as a character-layer node, not a hub or symbol.** Like `[[eld-cinderella-cycle]]`, `[[dsk-unlucky-lands-cycle]]`, `[[fear-of-cycle]]`, `[[castform-forms-cycle]]`, and `[[elemental-monkey-trio]]`, this is a collective conceptual entity hosted on the character layer per the Endriders precedent (`[[endriders]]`). A future `_cycles/` or `_rosters/` layer could re-home this entry alongside the other designer-confirmed cohort nodes when that split is commissioned; for now, single layer, same precedent.
- **Kasane Teto is UTAU, not Vocaloid.** Teto is canonically a UTAU voicebank (a distinct freeware singing-synthesizer ecosystem from Yamaha Vocaloid) who was joke-introduced as a fake Vocaloid in 2008 and has since been formalized as a beloved cross-ecosystem character. Her Project DIVA F appearance is licensed at the module / guest-character level. She is included in this PD/S22 cohort because the set licenses her PD/S22 module, but if a future BBL pass introduces a UTAU-specific node or a Crypton-strict Vocaloid node, Teto should be re-homed accordingly.
- **Yowane Haku originated as a fan-character.** Haku was originally a 2008 fan-character ("Crypton's failed Miku" / a sad-drunk-Miku-derivative joke); Crypton subsequently formalized her as a derivative fan-character and SEGA licensed her into Project DIVA as a guest module. Like Teto, her inclusion here is PD/S22-scope-appropriate but may be re-homed under a more granular Crypton-derivatives node if one is commissioned.
- **Module / costume variants are not separately enumerated.** Project DIVA F's module catalog contains hundreds of named costume variants per character. This node lists each card by its title (which names the module) but does not separately enumerate every module in the PD/S22 set. If a future bundle benefits from grouping by module (e.g. "every Hatsune Miku module that uses the celestial / divine register"), that grouping should be expressed at the bundle layer or as a sub-node, not by exploding this cohort node.
- **Many PD/S22 cards are non-character cards.** The PD/S22 set includes climax cards, song-event cards, item cards, and other non-character printings (e.g. *Continuing Dream*, *Time Machine*, *Glasses*, *Unhappy Refrain*) where the title is a song name or game event and the depicted character may or may not be a roster member. Those cards are not automatically members of this node; only cards whose primary subject is one of the named voicebank characters in a licensed appearance attach here. Edge-case adjudication (e.g. *Melancholic* the song-named card whose depicted figure IS Rin) is handled by the per-card vision pass and the card attaches if the IP attribution is canonical.
