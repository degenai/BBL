# Reference binders & inspiration

Visual references for what a well-curated Discrete Lair looks like when assembled in a physical binder. Drop images in this folder and add a one-line entry below.

The point: BBL's whole pitch hinges on lair buyers *seeing* curation as labor. These references are the visual answer to "why would I want this when I could just buy bulk by weight?"

## Index

- `misty-binder.png` — A Misty-themed Pokémon binder showcased on YouTube. Spans Misty's signature Pokémon (Psyduck, Staryu, Togepi, Lapras, Horsea), her trainer cards across multiple eras, and mixes Japanese + English printings. Exactly what a "Discrete Lair: Misty" looks like when someone takes the form seriously. Inspiration for character-anchor lairs across all card games.

## Adding a new reference

1. Drop the image in this folder. PNG or JPG, any size.
2. Add an entry above with: filename, one-sentence what-it-is, one-sentence why-it's-a-good-reference.
3. If the reference informs a curation mode or rule, link it from `../docs/curation-modes.md` or wherever the rule lives.
